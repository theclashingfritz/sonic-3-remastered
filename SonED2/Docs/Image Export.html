SonED2 - SonED2 Art Exporting


Most graphical data used in SonED2 can be exported to image files. All output files are 256-color TGA files of the sizes determined by the type and amount of data to export. These options appear in the "Misc" menu under "Export to image", according to the type of Project that is currently loaded:

  • Standard Level Projects
    • Tiles
    • Blocks
    • Chunks
    • Solidity
    • Level Maps

  • Sonic 1 Rotating Special Stage Projects
    • Layout Maps

    Standard Level Projects
      Tiles

        8x8 Tiles are dumped in vertical list format running in 4 columns, one for each palette row variation.


       Blocks

        16x16 Blocks are dumped in vertical list format, running 1 column.


       Chunks

        Chunks are dumped in a vertical list, running 2 columns.


       Solidity

        Solidity is dumped in a chart format running 20 entries wide, displaying each Height Map and its corresponding Angle value.


       Level Maps

        Level maps are dumped as a full image, as seen by scrolling through them in SonED2 or in-game. Either one of the two planes (A, foreground or B, background) can be dumped with only "High" priority tiles, only "Low" priority tiles, or both showing.

        Plane A dumps can also take advantage of the following additional output settings, which must be toggled within the drop-down menu prior to selecting Plane A for exporting:

          Include Solidity Path 0 - The map is overlaid with a visual representation of the solidity for Path 0. Cannot be used with "Include Solidity Path 1".
          Include Solidity Path 1 - The map is overlaid with a visual representation of the solidity for Path 1. Cannot be used with "Include Solidity Path 0".
          Include Angles In Solidity - The angle values of each of the Height Maps are overlaid onto the solidity
          Include Known Objects - All "Known" Objects (Objects with art defined within the Object Listing File) are drawn onto the map image, below the "High" plane. Cannot be used with "Include All Objects".
          Include All Objects - All Objects are drawn onto the map image, below the "High" plane. "Invisible" Objects, and Objects that are not defined within the Object Listing File are represented by a placeholder graphic. Cannot be used with "Include Known Objects".


     Sonic 1 Rotating Special Stage Projects
      Layout Maps

        Sonic 1 Special Stage Layout maps are dumped as a full image, as seen by scrolling through them in SonED2 or in-game.
