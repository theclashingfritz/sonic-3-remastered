SonED2 Sonic 1 Special Stage Editor


The Sonic 1 Special Stage Editor is used for editing the map layout and starting position data for Sonic 1's Rotating Special Stages.

The map layout is formed by placing "Objects" onto a 24x24 (pixel-measurement) grid. The images and descriptions displayed in the editor are set up through "Sonic 1 Special Stage Object Listing Files", which tell the editor how each object should be represented graphically. Those files may be altered and expanded upon as described by their own documentation.

This editor is capable of animating the "Objects", so that animated walls can easily be placed in the desired order.

This editor is made up of two editing modes:

Layout Editing Mode:
  • Status
  • Drawing to the layout
  • Selecting Object Type
  • Switch Editing Mode

Start Position Editing Mode:
  • Status
  • Moving a Player Start Position
  • Switch Editing Mode


Layout Editing Mode
    This mode is for editing the map layout of the Special Stage, and contains the following features:

     Status

      The Status Display appears at the upper-left corner of the editor and displays certain level and editor information:

      • Screen Position
        		•  - The X and Y world position of the top-left corner of the screen, shown in hex as with the in-game debug modes
      • Pointer Position
        		•  - The X and Y world position of the mouse cursor, shown in hex
      • Map Position
        		•  - The X and Y map layout grid position that the mouse cursor is currently hovering over
      • Current Object
        		•  - The ID of the "Object" that is currently selected for drawing to the map layout
      • Object At Cursor
        		•  - The ID of the "Object" that is currently placed at the map layout position under the cursor


    Drawing to the layout

      To draw the currently-selected "Object" onto the map layout, left-click the desired location, or hold the left mouse button and drag across the desired area


    Selecting Object Type

      To select the type of an "Object" by using the Sonic 1 Special Stage Object Type Selection List, double-left-click anywhere on the map layout. When the list appears, scroll to the description of the desired "Object" type and left-click. To exit the list without selecting an "Object" type, right-click instead

      To cycle between "Object" types using the keyboard, press the "A" or "Z" keys

      To "pick up" an "Object" type from the map layout, hover over an "Object" of the desired type and right-click


    Switch Editing Mode

      The "Editors" menu has options to switch between the two editing modes available to a project using this editor

      To switch to Start Position Editing mode using the keyboard, press "O"


Start Position Editing Mode
    This mode is for editing the player start locations for the Special Stage, and contains the following features:

     Status

      The Status Display appears at the upper-left corner of the editor and displays certain level and editor information:

        The status display always shows the following information:
      • Screen Position
        		•  - The X and Y world position of the top-left corner of the screen, shown in hex as with the in-game debug modes
      • Pointer Position
        		•  - The X and Y world position of the mouse cursor, shown in hex
        The following status elements are only available when the cursor is hovering over a Player Start Position:
      • Object Position
        		•  - The X and Y world position of the selected Player Start Position
      • Player Start Position ?
        		•  - The ID number of the selected Player Start Position


    Moving a Player Start Position

      To move a Player Start Position, click and drag the graphic representing that Start Position to its new location


    Switch Editing Mode

      The "Editors" menu has options to switch between the two editing modes available to a project using this editor

      To switch to Layout Editing mode using the keyboard, press "L"
